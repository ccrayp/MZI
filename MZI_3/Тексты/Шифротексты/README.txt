В текущей директории находится четыре текстовых файла

Кардано.txt - файл с шифротекстом, полученный шифрованием решёткой Кардано
Цезарь.txt константа - файл с шифротекстом, полученный шифрованием шифром Цезаря с константным ключем (17)
Цезарь.txt поговорка - файл с шифротекстом, полученный шифрованием шифром Цезаря с ключем поговоркой (Пустой колос голову кверху носит.)
Цезарь.txt ПСП - файл с шифротекстом, полученный шифрованием шифром Цезаря с ключем ПСП (инициализирующий вектор 01010101)
