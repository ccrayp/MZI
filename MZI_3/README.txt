Карта проекта

Гистограммы - директория с гистограммами текстовых файлов (открытого текста и полученных шифротекстов)
Исполняемые файлы - директория с файлами для запуска приложения (*.exe)
Исходные файлы - директория файлами исходного кода на C++ и файлы решения для открытия кода приложения
Тексты - директория с текстовыми файлами (открытого текста и полученных шифротекстов)
Отчет - документ в формате .docx, отчет по выполненной работе
