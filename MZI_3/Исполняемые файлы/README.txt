Частотный_анализатор.exe - приложение с графическим интерфейсом для частотного анализа текстовых и графических файлов (*.txt / *.bmp)
TraditionalMethod.exe - консольное приложение для шифрования Цезарем и решёткой Кардано, файл для запуска вручную
Cardano.exe - графический интерфейс для шифрования решёткой Кардано, запускается автоматически (скрытый файл)
out.txt - файл с сохраненными в процессе работы приложения данными (зашифрованный / расшифрованный текст)
